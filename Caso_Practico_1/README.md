# Caso Práctico #1 — Paquete de entrega

| Archivo | Descripción |
|---------|-------------|
| `caso_practico_1.md` | Enunciado completo + rúbrica + política anti-plagio. **Leer primero.** |
| `eventapp.zip` | Proyecto Spring Boot base (Nivel 2: READ ya funcionando). |

## Resumen rápido

- **Valor:** 15% de la nota final.
- **Modalidad:** individual, sincrónico durante la clase + extensión.
- **Duración en clase:** 3 horas.
- **Fecha de entrega:** la indica el profesor en la sesión.
- **Entrega final:** URL del repo + hash del commit, vía Moodle.

## Cómo empezar

1. Leer `caso_practico_1.md` **completo** antes de tocar código.
2. Verificar la rúbrica para entender cómo se califica.
3. Descomprimir `eventapp.zip` dentro de tu repo personal en una carpeta `Caso_Practico_1/`.
4. Seguir las instrucciones del `README.md` del proyecto para arrancarlo.
5. Implementar los requisitos R1 a R8.

## Si tenés dudas

Durante la clase, levantá la mano. Después de la clase, usar canal de Teams o consulta semanal.
